## ✨ Features

- **Performance monitoring** – real‑time CPU, RAM, disk, and network usage.
- **Remote power control** – shutdown or restart the server with one click.
- **Script & file execution** – run custom commands and scripts directly from the panel.
- **File manager** – browse, upload, delete, and edit server files.
- **Lightweight** – minimal resource footprint.
