**BrewSpherix** is a browser‑based remote server management tool.  
It gives system administrators a clean, real‑time dashboard to monitor performance, control server power, execute scripts, and manage files – all from any modern browser.

---

## ⚠️ Important

If you download a pre‑compiled build (`.exe`), Windows may flag it as a keylogger and VirusTotal as potentially dangerous. **This is NORMAL** – honestly, I don’t know why it happens. The project is in early alpha. If you don’t trust the build, you can compile it yourself from source.

---

## ✨ Features

- **Performance monitoring** – real‑time CPU, RAM, disk, and network usage.
- **Remote power control** – shutdown or restart the server with one click.
- **Script & file execution** – run custom commands and scripts directly from the panel.
- **File manager** – browse, upload, delete, and edit server files.
- **Lightweight** – minimal resource footprint.
- **Extensible** – designed to be easily extended with new modules.

---

## 🎯 Quick Start

### Prerequisites

- Python 3.8+
- pip

```bash
# Clone the repository
git clone https://github.com/Dalnoboihik/brewspherix.git

# Navigate to the project
cd brewspherix

# Install dependencies
pip install psutil

# Run the panel
python server.py
```
